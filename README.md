# 🛒 E-Commerce Website

A full-stack E-Commerce web application developed to provide an online shopping experience with product browsing, cart management, and secure user interaction.

## 📌 Features

- User Registration & Login
- Product Listing
- Product Search
- Shopping Cart
- Order Management
- Responsive Design
- Admin/Product Management
- Database Integration

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

### Backend
- PHP

### Database
- MySQL

### Server
- XAMPP
